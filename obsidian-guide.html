什麼是 Obsidian？

Obsidian 是一個筆記軟體，但跟 Word 或 Google Docs 很不一樣：

  • 所有筆記都存在你自己的電腦裡，是純文字檔案（.md 檔）
  • 筆記跟筆記之間可以互相連結，就像網頁上的超連結一樣
  • 你的筆記永遠是你的，不會被鎖在某個雲端服務裡
想像一下：你有一本超強筆記本——每一頁都可以跳到其他相關的頁面，而且永遠不會弄丟。
1

安裝 Obsidian

  1. 打開瀏覽器，前往 obsidian.md
  2. 點「Download」，下載 Windows 版本
  3. 執行安裝檔，一路按下一步就好
  4. 打開 Obsidian，你會看到歡迎畫面
  5. 選擇「建立新的 Vault」
  6. 名稱輸入：學習筆記
  7. 存放位置選擇你的 OneDrive 資料夾
  8. 按下「建立」
確認成功：你會看到一個乾淨的 Obsidian 畫面，左邊是空的檔案列表。
重要：vault 一定要建在 OneDrive 資料夾裡面，這樣學校和家裡的電腦才會自動同步。
2

認識介面

Obsidian 的畫面分成兩個主要區域：

┌──────────┬─────────────────────────────┐ │ │ │ │ 檔案 │ 編輯區 │ │ 列表 │ （你寫筆記的地方） │ │ │ │ └──────────┴─────────────────────────────┘
  • 左邊：你的所有筆記和資料夾（跟檔案總管很像）
  • 中間：寫筆記的地方

兩件事記住就好：

動作方法
建立新筆記按 Ctrl + N
存檔不用！Obsidian 會自動儲存
沒錯：完全不需要按 Ctrl + S，寫了就自動存好了。
3

學會基本格式（Markdown）

Obsidian 用一種叫 Markdown 的格式來寫筆記。聽起來很技術，但其實超簡單。你現在只需要學 5 個東西：

# 大標題 ## 小標題 - 重點一 - 重點二 **粗體強調**

就這樣，沒了。一個一個看：

標題

在文字前面加 # 和一個空格，就變成標題：

# 這是大標題 ## 這是小標題 ### 這是更小的標題

# 越多，標題越小。通常用 # 和 ## 就夠了。

列表

在文字前面打 - （減號加一個空格），就會變成列表：

- 第一個重點 - 第二個重點 - 第三個重點

粗體

用兩個星號把文字包起來，就會變粗體：

這裡有一個**很重要**的觀念

顯示出來會是：這裡有一個很重要的觀念

看看效果

按 Ctrl + E 可以在「編輯模式」和「閱讀模式」之間切換：

  • 編輯模式：你會看到 # 和 ** 這些符號
  • 閱讀模式：符號消失，顯示漂亮的排版效果
試試看：切換看看，感受一下差別！
4

建立你的資料夾結構

就像整理書包一樣，我們先把資料夾建好：

學習筆記/ ├── 國文/ ├── 英文/ ├── 數學/ ├── 自然/ ├── 社會/ └── 電腦/

怎麼建資料夾：

  1. 在左邊的檔案列表空白處，按右鍵
  2. 選「新增資料夾」
  3. 輸入資料夾名稱（例如 國文）
  4. 按 Enter

重複 6 次，把上面的資料夾都建好。

建完之後，左邊應該會出現 6 個資料夾，就像你書桌上的 6 個分類資料夾一樣。
5

寫你的第一篇筆記

來實際寫一篇筆記吧！跟著做：

  1. 按 Ctrl + N，建立新筆記
  2. 最上面的標題打：數學_比與比值
  3. 把這篇筆記拖進左邊的「數學」資料夾（用滑鼠拖放就好）
  4. 開始寫內容！

你可以參考下面這個範本：

# 比與比值 ## 今天學了什麼 - 比的定義：兩個數量的比較 - 比值 = 前項 ÷ 後項 - 比可以化簡（跟分數一樣） ## 重要公式 - a：b 的比值 = a/b ## 例題 - 3：5 的比值 = 3/5 = 0.6 - 6：4 的比值 = 6/4 = 1.5 ## 還不懂的地方 - 為什麼比值可以是小數？ - 比跟分數到底差在哪裡？
小提醒：「還不懂的地方」這一區超好用。下次上課前先看一眼，就知道要特別注意聽哪裡。
6

連結筆記（Obsidian 的超能力）

這是 Obsidian 最厲害的功能——筆記之間可以互相連結。

怎麼做

在筆記裡打兩個左方括號 [[，然後輸入另一篇筆記的名稱，再打兩個右方括號 ]]：

今天學的比與比值，跟之前學的 [[數學_分數運算]] 有關。

會發生什麼

  • [[數學_分數運算]] 會變成一個可以點擊的連結
  • 點下去就會跳到那篇筆記
  • 如果那篇筆記還不存在，Obsidian 會用虛線顯示，點下去會自動幫你建立

為什麼這很厲害

想像你在寫自然筆記「密度」的時候，發現需要用到數學的「比與比值」：

密度 = 質量 ÷ 體積，其實就是一種 [[數學_比與比值]]
重點：這樣你的知識就不再是一科一科分開的，而是互相連起來的網路。這就是 Obsidian 跟一般 Word 檔或紙本筆記最大的不同。
7

每天的使用流程

每天只需要花 5 分鐘：

  1. 打開 Obsidian
  2. Ctrl + N 建立新筆記
  3. 標題格式：科目_主題（例如：英文_現在完成式、自然_密度）
  4. 拖進對應的資料夾
  5. 寫筆記，用 # 分段、- 列重點
  6. 有關聯的就打 [[連結]] 連過去
就這樣，不需要做更多。先養成習慣最重要。
!

小挑戰

今天回家後，請完成以下任務：

  • 挑一科，寫一篇課堂筆記
  • 筆記裡要用到：# 標題、- 列表
  • 至少打一個 [[連結]]（連到任何你覺得相關的主題）

不用寫很多，半頁就好。重點是動手做一次！

Markdown 速查表

把這張表記起來，寫筆記就夠用了：

你打的文字顯示效果說明
# 標題大標題井字號 + 空格
## 小標題小標題兩個井字號 + 空格
- 項目列表圓點減號 + 空格
**粗體**粗體兩個星號包起來
[[筆記名]]可點擊的連結兩個方括號包起來
之後等你用習慣了，我們再學更多功能。現在先把這些練熟就好！
