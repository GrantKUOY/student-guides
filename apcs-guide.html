Phase 1（第 2–6 週）

打穩基礎：串列 + 函式 + 字串

這個階段是拿二級分數的關鍵，也是三級的地基。每個主題學完就去 ZeroJudge 找對應題目驗收。

A

串列 List（最重要，第 2–3 週）

APCS 幾乎每題都用到串列，一定要紮實
為什麼串列這麼重要？ 串列就是「有順序的資料盒子」。考試題目常常要你處理一堆數字或字元，沒有串列就沒辦法存。

必學語法

Python 語法
# 建立串列 nums = [3, 1, 4, 1, 5, 9] # 讀取元素（從 0 開始算） print(nums[0]) # 3 print(nums[-1]) # 9（最後一個） # 修改元素 nums[0] = 100 # 常用方法 nums.append(2) # 在末尾加入 2 nums.remove(1) # 移除第一個 1 print(len(nums)) # 長度 print(min(nums)) # 最小值 print(max(nums)) # 最大值 nums.sort() # 排序（由小到大） # 用 for 遍歷 for x in nums: print(x) # 用 for 加索引 for i in range(len(nums)): print(i, nums[i])

練習題目（依序完成）

  • P1：輸入 5 個數字存入串列，輸出最大值與最小值
  • P2：輸入 N 個成績，計算平均、最高、最低分
  • P3：輸入 N 個數字，反轉後輸出（不能用 .reverse()，要自己寫）
  • P4：輸入 N 個數字，找出第二大的數
  • P5：輸入一串數字，找出所有比平均值大的數

B

函式 Function（第 4–5 週）

讓程式碼可以重複使用，是寫長程式的關鍵
為什麼要學函式？ APCS 的題目通常需要把大問題拆成小問題來解。函式讓你把每個小問題包裝起來，分開處理、分開測試。
Python 語法
# 定義函式 def add(a, b): return a + b # 呼叫函式 result = add(3, 5) print(result) # 8 # 把你之前寫過的程式重構成函式 def is_prime(n): if n < 2: return False for i in range(2, n): if n % i == 0: return False return True def find_divisors(n): result = [] for i in range(1, n+1): if n % i == 0: result.append(i) return result
  • 把你一上寫過的質數判斷改成 is_prime(n) 函式版本
  • 把因數找尋改成 find_divisors(n) 函式版本
  • 把 GCD 改成 gcd(a, b) 函式版本
  • 組合三個函式，做一個「數學小幫手」：輸入一個數，自動輸出因數、是否質數、因數總和

二維串列（第 7–8 週）

把串列放進串列，就是矩陣/表格
Python 語法
# 建立 3x3 矩陣 matrix = [ [1, 2, 3], [4, 5, 6], [7, 8, 9] ] # 讀取元素：matrix[列][欄] print(matrix[0][0]) # 1（第0列第0欄） print(matrix[1][2]) # 6（第1列第2欄） # 用巢狀迴圈遍歷 for row in matrix: for val in row: print(val, end=" ") print() # 換行 # 建立 N×M 全零矩陣 n, m = 3, 4 grid = [[0]*m for _ in range(n)]
  • P：建立 3×3 矩陣，計算各列總和與各欄總和
  • P：輸入 N 位同學 M 科成績，計算每人平均、每科平均
  • P：找出矩陣中最大值，並輸出它在第幾列第幾欄
E

排序演算法（第 9 週）

不只是會用 .sort()，要能自己寫出來
為什麼要自己寫排序？ APCS 觀念題常常考「這段程式碼執行後，陣列會變成什麼？」你必須懂排序的每一步在做什麼，才能追蹤結果。

泡沫排序（Bubble Sort）

Python 語法
# 泡沫排序：每次把最大的泡泡「浮」到右邊 def bubble_sort(arr): n = len(arr) for i in range(n): for j in range(n - i - 1): if arr[j] > arr[j+1]: arr[j], arr[j+1] = arr[j+1], arr[j] return arr nums = [5, 3, 8, 1, 9, 2] print(bubble_sort(nums)) # [1, 2, 3, 5, 8, 9]

選擇排序（Selection Sort）

Python 語法
# 選擇排序：每次從未排序部分找最小值，放到前面 def selection_sort(arr): n = len(arr) for i in range(n): min_idx = i for j in range(i+1, n): if arr[j] < arr[min_idx]: min_idx = j arr[i], arr[min_idx] = arr[min_idx], arr[i] return arr
  • 手動追蹤泡沫排序：[5, 3, 8, 1] 每一輪結束後陣列長什麼樣？
  • 自己從頭打出泡沫排序（不看上面），跑一遍確認正確
  • 自己從頭打出選擇排序，跑一遍確認正確
  • 比較兩種排序的差異，用 Obsidian 寫下你的理解
F

二分搜尋（第 10 週）

在已排序的串列中快速找目標值
二分搜尋的概念 就像翻字典：先翻到中間，判斷目標在左半還是右半，每次排除一半範圍，很快就能找到。
Python 語法
def binary_search(arr, target): left, right = 0, len(arr) - 1 while left <= right: mid = (left + right) // 2 if arr[mid] == target: return mid # 找到了，回傳索引 elif arr[mid] < target: left = mid + 1 # 目標在右半 else: right = mid - 1 # 目標在左半 return -1 # 找不到 nums = [1, 3, 5, 7, 9, 11] print(binary_search(nums, 7)) # 3（索引） print(binary_search(nums, 4)) # -1（找不到）
  • 手動追蹤：在 [1,3,5,7,9,11] 中找 9，每一步 left/mid/right 是多少？
  • 自己從頭打出二分搜尋，測試幾個案例
  • 延伸練習：找「第一個大於等於目標的位置」
G

Stack / Queue 基礎（第 11 週）

用串列模擬這兩種資料結構
Python 語法
# Stack（堆疊）：後進先出（LIFO） stack = [] stack.append(1) # push stack.append(2) stack.append(3) print(stack.pop()) # 3（最後放的先出來） # Queue（佇列）：先進先出（FIFO） from collections import deque queue = deque() queue.append(1) # enqueue queue.append(2) queue.append(3) print(queue.popleft()) # 1（最早放的先出來）
  • 用 Stack 實作：判斷括號是否對稱，例如 "((()))" 是對稱，"(()" 不是
  • 用 Queue 模擬：銀行排隊，依序服務顧客

如何用 Claude 輔助 APCS 學習

Claude 在 APCS 學習中可以扮演幾個不同的角色。

角色一：概念講解員

貼給 Claude（學新概念）
我是國中生，正在學 Python 的串列（list）。 請用以下方式教我： 1. 用生活中的例子解釋串列是什麼 2. 給我 5 行最常用的語法，並說明每行在做什麼 3. 出一個簡單的練習題讓我試試看

角色二：除錯助手

重要原則 先自己想至少 10 分鐘，再請 Claude 幫忙。告訴 Claude 你已經試過什麼、你覺得哪裡有問題，不要直接說「幫我寫這題」。
貼給 Claude（除錯）
我在寫 APCS 練習題，程式碼如下： （貼上你的程式碼） 輸入：（貼上測試用的輸入） 我期望的輸出：（你覺得應該輸出什麼） 實際輸出：（程式實際輸出了什麼） 我已經試過：（說明你做過哪些嘗試） 我猜問題在：（說明你的猜測） 請幫我找出問題，但不要直接給我完整的解答，先給我提示就好。

角色三：觀念題考官

貼給 Claude（觀念測驗）
請出 3 題 APCS 觀念題（選擇題）風格的題目， 難度大約是二到三級，主題是「串列與迴圈」。 一次出一題，等我回答後再出下一題，並告訴我對不對。

角色四：演算法解說員

貼給 Claude（演算法理解）
請用純文字（不要用圖）一步一步說明泡沫排序（Bubble Sort） 對陣列 [5, 3, 8, 1] 的每一步操作。 我是國中生，請說得很仔細，讓我能用手追蹤每一步。
