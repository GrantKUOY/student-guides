事前確認

你的電腦需要是 Windows 10（版本 2004 以上） 或 Windows 11。

怎麼檢查版本：

  1. 按鍵盤 Win + R（Win 就是那個 Windows 旗幟鍵）
  2. 輸入 winver，按 Enter
  3. 看到版本號 ≥ 2004 就可以
版本太舊？先去 Windows Update 更新系統。
0

開啟 PowerShell（系統管理員）

你在做什麼：用管理員權限開啟命令列
  1. 點左下角「開始」按鈕，搜尋 PowerShell
  2. 對「Windows PowerShell」按右鍵
  3. 選「以系統管理員身分執行」
  4. 如果跳出詢問視窗，點「是」
確認：視窗最上方標題列有「系統管理員」字樣。
1

安裝 WSL

你在做什麼：安裝 Linux 子系統

在 PowerShell 視窗中貼上：

PowerShellwsl --install -d Ubuntu

按 Enter，等它跑完。

確認：看到「已成功安裝發佈」或 "The requested operation is successful"。
接下來要重開機。存好你正在做的事，然後重新啟動電腦。
2

設定 Linux 帳號密碼

你在做什麼：建立你的 Linux 使用者

重開機後，Ubuntu 視窗會自動跳出來（如果沒有，從開始選單找「Ubuntu」打開）。

  1. 它會問你 Enter new UNIX username:，輸入一個英文小寫的帳號名（例如你的名字拼音）
  2. 它會問你 New password:，輸入一個你記得住的密碼
  3. 它會要你再輸入一次密碼確認
密碼打字時螢幕完全不會顯示任何字元，沒有星號、沒有圓點，什麼都沒有。這是正常的！不是鍵盤壞了。照打就好，打完按 Enter。
確認：看到 $ 符號的提示字元，像這樣：
yourname@DESKTOP-XXXX:~$
3

更新系統

你在做什麼：把 Linux 系統更新到最新

在 Ubuntu 視窗中貼上：

Ubuntusudo apt update && sudo apt upgrade -y

它會要你輸入步驟 2 設的密碼（一樣打字不會顯示，照打按 Enter）。

等它跑完，大概 1-3 分鐘。

確認：跑完後最後幾行沒有出現 error，回到 $ 提示字元。
4

安裝 Node.js

你在做什麼：安裝 Claude Code 需要的執行環境
不要用 sudo apt install nodejs！那個版本太舊，Claude Code 會裝不起來。請照下面的方式安裝。

動作 4-1：先加入 Node.js 官方套件來源：

Ubuntucurl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -

動作 4-2：安裝 Node.js：

Ubuntusudo apt install -y nodejs

確認：檢查版本：

Ubuntunode --version
確認：應該顯示 v20.x.x 或更高。Node.js 裝好了，剩兩步就完成。
5

安裝 Claude Code

你在做什麼：安裝 AI 程式助手
Ubuntunpm install -g @anthropic-ai/claude-code

等它跑完，大概 1-2 分鐘。

確認：

Ubuntuclaude --version
確認：顯示版本號，例如 1.x.x。
6

註冊 Anthropic 帳號

你在做什麼：取得 AI 的使用權限
  1. 打開瀏覽器，進入 console.anthropic.com
  2. 點 Sign up 註冊帳號（可以用 Google 帳號直接登入）
  3. 照畫面指示完成註冊
確認：能登入 console.anthropic.com 的主畫面。
好消息：新帳號有 $5 美元免費額度，正常使用夠用好幾個月。

第一次啟動 Claude Code

你在做什麼：登入並啟動 AI 助手

在 Ubuntu 視窗中輸入：

Ubuntuclaude

畫面會出現登入選項，選擇第 2 項（Anthropic Console account），然後照畫面指示完成登入。過程中會跳出瀏覽器要你授權，點同意就好。

確認：看到 Claude Code 的對話介面，游標在等你輸入。安裝全部完成！

在 VS Code 裡使用 Claude Code

你在做什麼：讓你可以一邊看程式碼、一邊跟 AI 對話

9-1｜安裝 VS Code

到 code.visualstudio.com 下載安裝。

9-2｜安裝 WSL 擴充套件

  1. 打開 VS Code
  2. 按 Ctrl + Shift + X 開啟擴充套件面板
  3. 搜尋 WSL
  4. 安裝由 Microsoft 出的「WSL」擴充套件

9-3｜安裝 Claude Code 擴充套件

  1. 同樣在擴充套件面板搜尋 Claude Code
  2. 安裝由 Anthropic 出的「Claude Code」擴充套件（注意看發行者名稱，選官方的）

9-4｜用 VS Code 連到 WSL

  1. 按 Ctrl + Shift + P 開啟命令面板
  2. 輸入 WSL: Connect to WSL
  3. 選擇 Ubuntu
  4. VS Code 左下角會顯示 WSL: Ubuntu，表示連線成功

9-5｜開啟 Claude Code 面板

點 VS Code 右上角的火花圖示，或按 Ctrl + Shift + P 搜尋 Claude Code。

┌──────────┬──────────────────┬──────────────┐ │ │ │ │ │ 檔案 │ 程式碼編輯區 │ Claude │ │ 總管 │ （寫 Python） │ Code 對話 │ │ │ │ │ └──────────┴──────────────────┴──────────────┘
這就是你以後寫 APCS 的工作環境！
*

每次使用

方法一：VS Code（推薦）

  1. 打開 VS Code
  2. 左下角確認連到 WSL: Ubuntu（如果沒有，按 Ctrl + Shift + P → WSL: Connect to WSL）
  3. 左邊開檔案、中間寫程式、右邊問 Claude

方法二：純終端機

  1. 從開始選單打開 Ubuntu
  2. 輸入 claude
  3. 直接在終端機裡對話

錯誤急救表

你看到的原因解法
exec: node: not foundNode.js 沒裝好重做步驟 4
npm ERR!通常是網路問題確認有連網，再執行一次步驟 5
密碼打不出來正常的！Linux 打密碼不顯示照打，按 Enter
wsl --install 沒反應沒用系統管理員開回步驟 0，確認用管理員身分
Ubuntu 視窗閃退WSL 沒裝完就關了重開機，再從開始選單開 Ubuntu
EACCES permission denied權限不足在指令前面加 sudo
VS Code 左下角沒有 WSL: Ubuntu沒連到 WSLCtrl+Shift+P → WSL: Connect to WSL
Claude Code 面板沒反應擴充套件沒在 WSL 裡安裝連到 WSL 後，重新安裝 Claude Code 擴充套件
